¡Bienvenido al proyecto Dcode!(Un proyecto basado en el sistema de Cifrado Cesar)Este proyecto está dirigido a personas que desean intercambiar mensajes secretos utilizando un método de cifrado simple pero efectivo. El cifrado César es una técnica de cifrado por sustitución en la que cada letra del mensaje original se desplaza un número fijo de posiciones (que llamaremos “offset”) hacia la derecha en el alfabeto. Por ejemplo, si el número fijo es 3, la letra "A" se cifrará como "D", la letra "B" como "E", y así sucesivamente.Para utilizar este programa, siga los siguientes pasos:1. Abra su navegador favorito y pegue las url de nuestra webhttp://localhost:3000/2. Presione el boton con la opcion CIFRAR para cifrar un mensaje o el boton DESCIFRAR para descifrar un mensaje.3. Ya en la pagina de su eleccion ingrese el mensaje que desea (tomaremos como ejemplo el boton CIFRAR ) cifrar y el número de desplazamiento deseado.4. El programa cifrará el mensaje utilizando el cifrado César y lo mostrará en la pantalla.5. Comparta el mensaje cifrado con su destinatario.6. El destinatario puede utilizar el mismo programa para descifrar el mensaje utilizando el número de desplazamiento correcto (el mismo que se utilizo para el cifrado).¡Recuerde no compartir el número de desplazamiento con nadie más que su destinatario para asegurar la privacidad de su mensaje!Esperamos que este proyecto le sea útil en sus intercambios de mensajes secretos. Si tiene alguna pregunta o comentario, no dude en ponerse en contacto con nosotros. ¡Gracias por utilizar nuestro programa!
